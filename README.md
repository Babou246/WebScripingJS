# WebScripingJS
### Pour la Partie JS
### Mettez-vous sur le repertoire static/

### Taper:
npm install 

### Créer l'environnement virtuel
python3 -m venv <nom_environnement> '<br>'
source env/bin/<nom_environnement>

### Telecharger les dépendences 
pip install -r requirements.txt

### lancer le serveur 
python3 main.py
